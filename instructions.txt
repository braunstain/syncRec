SYNC STEPS
==================
1. install chrony - NTP clock sync service
sudo apt update
sudo apt install chrony

2. set Pi1 as server via /etc/chrony/chrony.config
add
allow 192.168.0.101/24 (for each allowed client)
local stratum 8

3. set Pi2 as client via /etc/chrony/chrony.config
comment/delete # pool 2.debian.pool.ntp.org iburst
add
server 192.168.0.100 iburst

4. restart service on all devices

5. check all clients are using configured server as host with:
chronyc sources -v
check output has:
^* 192.168.0.100 (server's_ip)

OPTION A - RECORD WITH SOUNDDEVICE (PYTHON LIBRARY)
===========
make sure all clients has /home/mic1/record_precise.py

1. on each client:
sudo apt install libportaudio2
pip install sounddevice --break-system-packages
sudo apt install python3-scipy python3-numpy

2. run on laptop run_test.ps1 (test results folder will be created on ./test_results

to analyze test results:

python analyze_sync.py
